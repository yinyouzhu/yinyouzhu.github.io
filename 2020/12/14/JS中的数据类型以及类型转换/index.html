JS中的数据类型以及类型转换

JS基础知识
JS基础知识
发布日期:   2020-12-14
文章字数:   861
阅读时长:   3 分

JS中的数据类型有几种？有人说是有6种，最近发现很多人对JS的基础不太了解，所以就写一篇笔记来总结一下吧，如不对的地方请多多指教！万分感谢

首先需要说明的是，JS中的数据类型是分为基本数据类型和引用数据类型,其中，基本数据类型也可以称为值类型。

基本数据类型（值类型）

Number(数字)

NaN是一种特殊的数字类型，NaN和谁都不相等，包括他自己。需要用isNaN( )方法检测,如果是数字则返回false，非数字返回true
isNaN(123)   -->  false
isNaN('test')  -->  true
isNaN(undefined)  --> true
isNaN(NaN)   ---> true
isNaN(null)  --> false
把其他的数据类型转换为number的时候，如果不能转化，则返回NaN(使用number()进行隐式转换)
Number('123')  -->  123
Number('test')  -->  NaN
Number(NaN)   -->  NaN
Number('999 888')  --> NaN 因为999和888之间有一个空格，所以无法转化
特殊 Number('') --> 0 --> false
isNaN检测机制
首先检测当前要检测的值是否为数字类型，如果不是，浏览器会默认把值转换为数字类型

1. 其他基本类型转换为数字，直接使用Number()方法
2. 布尔类型数字转数字 Number(true) --> false
3. 其他类型 Number(null)  --> false
           Number(undefined) --> NaN --> false
引用类型转换为数字
先使用toString()转化成字符串，然后再调用Number()转换为数字
[12,23] -> toString() -> '12,23' -> Number() -> NaN -> true
[12] -> toString() -> '12' -> Number() -> false
对象转化为数字
对象转化为数字，应该先valueOf,没有原始值再toString变为字符串，最后把字符串转化为数字

例:({name:'xiaoming'}).valueOf() --> {name:'xiaoming'}.toString() --> [object object] 然后Number([Object Object])  --> NaN --> true
注: 对象{}.toString得到的是[Object Object],调用原型链Object.prototype.toString()

string (字符串)

字符串字面量用以下形式展现
'string'
"string"
"中文"
"日本語"

boolean （布尔）

0 null undefined ‘’ NaN 这五个值转化为boolean都是false，其余的都是true

其他数据类型都可以转化为boolean类型
如何转化？
Boolean(需要转化的值)进行转化
!!() // 两次取反，相当于没取反，等价于Boolean()
!()  // 先转化成boolean类型，然后再取反
!(0) -->true

null （对象空指针）

对象空指针(意料之中的没有，一般是人为手动，先赋值null，后面再进行赋值操作)

undefined （未定义）

未定义(天生没有，一般都不是人为手动控制，大部分是浏览器自主控制)

symbol （唯一值）

应用场景：

  1. 给对象设置唯一属性
  2. 给vuex/redux中做行为派发，统一管理派发的行为标识，标识的值可以是唯一值
Symbol() === Symbol() --> false  
直接使用Symbol()就可以创建新的Symbol类型 
var symValue1 = Symbol();
var symValue2 = Symbol('foo')
var symValue3 = Symbol('foo')
symValue2 === symValue3 --> false

bigint(超大数字)

bigInt是一个内置对象，它提供了一种方法来表示大于2的53次方-1的整数。

可以用在一个整数字面量后面加n的方式定义一个bigInt,如10n，或者调用函数BigInt() 

let bigIntValue = 200315654n
let alsoHuge = BigInt(3365422154);

在使用typeof测试时，bigInt对象返回 bigint 
let value1 = BigInt(123);
typeof(value1)   --> bigint

引用数据类型

对象数据类型 (object)

其中包括：普通对象、实例对象、数据对象、正则对象、prototype原型对象

函数数据类型 (function)

其中包括：日期对象
